Sales Soldier Setup

1. Install Python on the new computer.
2. Open this folder.
3. Double-click Install_Sales_Soldier.bat to install required Python packages.
4. Open the software using Sales Soldier.lnk, Sales Soldier.vbs, or Run_Sales_Soldier.bat.

Included files:
- gstr1_tool.py
- requirements.txt
- Install_Sales_Soldier.bat
- Install_SGST.bat
- Run_Sales_Soldier.bat
- Sales Soldier.vbs
- Sales Soldier.lnk
- GSTR1_Template.xlsx
- master.xlsx

Notes:
- GSTR1_Template.xlsx is the base transfer template.
- master.xlsx contains the saved party and product masters.
- Export Excel is the regular monthly invoice export.
- Export GSTR-1 Excel creates the GST-ready structured workbook.
