This is a project designed for CA Students who are intended to check the batch details of MCS course only. 
It is advised for them to update an excel file with branches [ Exact branch names as per the website ] and save it. It is also recommended to have a separate column named City where you can fill the cities for which MCS batch availability are to be requested. You can see the sample excel file uploaded.
It is also advised to install the required packages before running the code
